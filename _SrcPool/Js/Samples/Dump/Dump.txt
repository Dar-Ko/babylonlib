Install Microsoft Script Debugger:
  To enable it, open IE and go to Tools|'Internet Options'|Advanced and uncheck "Disable Script Debugging (Internet Explorer)".
  If the debugger cannot debug any script code in the application you are trying to debug, delete or rename the KLM\Software\Microsoft\Ole\DefaultAccessPermission key.
