***TextWrap*** plugin for the [Pluma text editor](https://github.com/mate-desktop/pluma "Pluma v1.26+") (MATE 24.04 desktop) enables a quick toggle for text wrapping, which is a common limitation in the default editor..Â 

- Tested with Ubuntu MATE 24.04.01 and Pluma 1.26.1
- **Installation:** Download `TextWrap.plugin` and `TextWrap.py` and placing them in the appropriate *Pluma* plugins directory (usually `~/.local/share/pluma/plugins/`).
- **Usage:** Once enabled in *Edit > Preferences > Plugins*, it allows users to quickly toggle line wrapping, often with a keyboard shortcut.
