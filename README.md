Babylon Library is a collection of simple routines and classes that could be used as building blocks of larger projects. Many of the methods are portable and adjusted for use in embedded systems.

* * *

- [Babylon Library scripts, configuration files](Util) for Linux and Windows.

### Plugins for Pluma, the MATE text editor

- [*TextWrap*](https://github.com/Dar-Ko/babylonlib/tree/master/_SrcPool/Py/Pluma "TextWrap for Pluma 1.26+") plugin is a quick shortcut for wrapping text, addressing the lack of a default toggle.
