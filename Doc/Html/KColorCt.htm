4-bit Color Table

Video Graphics Array (VGA) is a computer display standard defined in 1987 by IBM. VGA has several display modes and one of the most used has 16 colors and size of 640×480 pixels.

Value (RGB) Hexadecimal HTML Color
0 0 0 0 #000000 black Black
1 0 0 128 #000080 navy Blue
2 0 128 0 #008000 green Green
3 0 128 128 #008080 teal Cyan
4 128 0 0 #800000 maroon Red
5 128 0 128 #800080 purple Magenta
6 128 128 0 #808000 olive Brown
7 200 200 200 #C8C8C8 silver White
8 128 128 128 #808080 gray Grey
9 0 0 255 #0000FF blue Light Blue
10 0 255 0 #00FF00 lime Light Green
11 0 255 255 #00FFFF aqua Light Cyan
12 255 0 0 #FF0000 red Light Red
13 255 0 255 #FF00FF fuchsia Light Magenta
14 255 255 0 #000000 yellow Yellow
15 255 255 255 #FFFFFF white Bright White
Table 1The 16 colors of VGA text mode.

Note: The RGB value of silver color used in HTML/CSS is LIGHTGREY = #C0C0C0.
